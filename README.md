# 課題提出

https://github.com/ak03120/10209323/blob/main/src/main/kotlin/Main.kt

bubbleSort()がバブルソートで、quickSort()がクイックソートです。

## バブルソートを選んだ理由

ソートという概念を習った際、初めて習ったのがバブルソートでした。何も知らない状態から初めてコードを動かせて喜びを感じたので、思い入れがありバブルソートを選択しました。

## クイックソートを選んだ理由

シングルスレッド処理で速いと言われているクイックソート、実用性のあるソートを1つは実装してみたいとの思いで選びました。
